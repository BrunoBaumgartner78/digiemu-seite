# DigiEmu (Application Layer)

This repository contains an application-layer implementation of DigiEmu (Beta / v1.1.2).

It represents product-level experimentation and deployment logic and is not part of the DigiEmu Core knowledge infrastructure.

## Scope

- Web application (Next.js)
- UI, authentication, and product logic
- Deployment configuration

## Relation to DigiEmu Core

DigiEmu Core is the auditable knowledge kernel and lives in a separate repository.

This project builds on higher-level application logic.
